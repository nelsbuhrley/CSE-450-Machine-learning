# CSE 450 — Machine Learning

Collaborative coursework for CSE 450 at BYU-Idaho. Classification pipelines, exploratory data analysis, and predictive modeling on real-world datasets.

## Modules

### [Module 2: Bank Marketing — Term Deposit Prediction](module_2-bank/)

Built three classifiers (Random Forest, RF + SMOTE, Stacking RF + KNN) to predict which bank clients will subscribe to term deposits. Includes cost-sensitive evaluation, SMOTE oversampling, threshold tuning, and a business-value framework that turns a money-losing campaign profitable.
